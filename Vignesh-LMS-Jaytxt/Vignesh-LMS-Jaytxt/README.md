# Textile Leave Management System

A comprehensive leave and attendance management system designed specifically for textile manufacturing companies.

## Features

### For Administrators
- **Dashboard Overview**: Real-time statistics and analytics
- **Employee Management**: Add, view, and manage employee profiles
- **Leave Management**: Approve/reject leave requests with comments
- **Attendance Tracking**: Mark and manage employee attendance
- **Reports & Analytics**: Generate detailed reports and charts
- **Multi-department Support**: Weaving, Spinning, Dyeing, Finishing, etc.

### For Employees
- **Self-service Portal**: Apply for leaves and check status
- **Attendance View**: View personal attendance records
- **Leave Balance**: Check available leave balances
- **Profile Management**: Update personal information
- **Notifications**: Receive updates on leave status

### System Features
- **Role-based Access Control**: Separate admin and user dashboards
- **Shift Management**: Support for textile industry shift patterns
- **Leave Types**: Annual, Sick, Casual, Emergency leaves
- **Holiday Calendar**: Manage company and national holidays
- **Responsive Design**: Works on desktop and mobile devices
- **Secure Authentication**: Password hashing and session management

## Technology Stack

- **Backend**: Flask (Python)
- **Database**: SQLite (with SQLAlchemy ORM)
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Bootstrap 5, Custom CSS
- **Charts**: Chart.js
- **Icons**: Font Awesome
- **Notifications**: Toastr
